Missão
Books Tinder é uma aplicação para procurar livros online, cruzando informações de preferência do usuário. Está disponivel em todas as plataformas: iOS, Android e Web.
Conceito
Um livro não se julga pela capa, assim o Book Tinder pretende perceber os gostos do usuário e sugerir livros que se adequem antes deste os comprar.
